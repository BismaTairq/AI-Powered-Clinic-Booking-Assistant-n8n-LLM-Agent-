# 🏥 AI-Powered Clinic Booking Assistant (n8n + LLM Agent)

## 🚀 Overview
An end-to-end AI-powered chatbot system designed for modern healthcare websites. This solution enables patients to book appointments, ask queries, and receive real-time assistance through a conversational interface.

Built using n8n workflow automation and LLMs (OpenAI / Google Gemini), the system delivers intelligent automation with seamless frontend-backend integration.

---

## 💡 Problem
Traditional clinic websites rely on manual processes:
- Users must browse services manually  
- Appointment booking requires calls or forms  
- Delayed responses lead to poor user experience  

---

## ✅ Solution
This project introduces an AI-driven conversational assistant that:
- Understands user intent (booking vs general queries)  
- Automates appointment scheduling  
- Provides instant AI-generated responses  
- Maintains conversation context using memory  

---

## ⚙️ Key Features

### 🤖 AI Chatbot Integration
- Powered by LLMs (OpenAI / Gemini)  
- Handles natural language conversations  

### 🔀 Intelligent Routing (Agent System)
- Router Agent detects user intent  
- Routes requests to:
  - 📅 Appointment Booking Agent  
  - 💬 Conversational Support Agent  

### 📅 Automated Appointment Booking
- Fetches available time slots  
- Books appointments dynamically  

### 🧠 Memory System
- Window buffer memory for context-aware conversations  

### 🌐 Custom Frontend Chat Widget
- Responsive UI built with HTML, CSS, JavaScript  
- Real-time interaction via webhook  

### 🔗 Webhook API Integration
- n8n webhook processes frontend requests  
- Returns structured JSON responses
